= Robot Value

The robot's value is part of her fabrication info. 
The value is usually random.
This value represents the base value relative to other robot types.
Peripherals and programmes increase the value of the robot.
The referee will find it a lot easier to stick with the base value.

Valuable robots, or robots with unique peripherals, will often find that they have "buyers" pursuing them.
Buyers usually do not understand that a persona robot has free will and is resistant to forcible purchase.
